# LWL(Low Weight Library)

## Description
LWL is *Low Weight Library* that mean C library that use when I need to develop SShell except what I don't need.
## Programming Languages
- C
- Assembly(**x86_64**)

## System Requirements
- OS: Linux for **x86_64**
- Compiler: **GCC**
